# File Archive

## Author
Acha Bill [achabill12@gmail.com]

## Functionality
- Upload files
- Download file (full or multipart)

## Usage
Add dependency in maven  
    
    <dependency>  
         <groupId>org.skylabase</groupId>
         <artifactId>file-archive</artifactId>
         <version>1.0</version>
    </dependency>
